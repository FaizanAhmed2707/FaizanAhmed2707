## C.A.S.E (Centralized Academic System for Excellence)
**Overview**  
*Duration*: January 2025 – June 2025  
*Team Size*: 3 members  
*Technologies*: React.js, TypeScript, Next.js, PostgreSQL, JWT, Framer Motion  

C.A.S.E is a cutting-edge web platform that revolutionizes academic and administrative management for educational institutions. By centralizing access for students, faculty, parents, and administrators, it ensures seamless, secure, and organized interaction with academic data. Imagine a single hub where students track attendance, faculty manage schedules, parents monitor progress, and admins oversee it all—powered by a sleek, role-based interface and an AI chatbot for instant query resolution.

**Key Features**:  
- **Role-Based Dashboards**: Tailored interfaces for students, faculty, parents, and admins.  
- **Core Modules**: Manage profiles, academic records, attendance, timetables, and semester-wise notes.  
- **AI-Powered Chatbot**: Enhances student experience with real-time query assistance.  
- **Secure Authentication**: Leverages JSON Web Tokens (JWT) for robust, role-specific access.  

**My Contributions**:  
As the team leader, I drove the vision and execution of C.A.S.E, ensuring a cohesive and impactful solution. My key contributions include:  
- **Feature Design**: Crafted functionalities for all user roles, detailed in the Software Requirements Specification (SRS).  
- **System Architecture**: Designed a scalable 3-tier model, supported by UML diagrams (Use Case, Class, Component, Deployment).  
- **Frontend Development**: Built dynamic, responsive interfaces using React.js, including login pages and dashboards.  
- **Security Implementation**: Integrated JWT for secure, role-based authentication and session management.  

**Outcomes**:  
C.A.S.E delivered a robust proof-of-concept, blending modern web technologies with software engineering principles. The platform’s modular design supports future enhancements like mobile apps and predictive analytics. Key achievements:  
- **90% Code Coverage**: Ensured reliability through rigorous testing.  
- **50% Faster Data Retrieval**: Optimized database queries for efficiency.  
- **Scalable Architecture**: Ready for institution-wide deployment.

## Used Car Price Predictor using Machine Learning
**Overview**  
*Duration*: August 2024 – October 2024  
*Team Size*: 1 member (Solo Project)  
*Technologies*: XGBoost, Streamlit, Python, Google Generative AI API  

The Used Car Price Predictor is an innovative tool that transforms how resale car prices are estimated. Say goodbye to subjective dealer appraisals! This project combines a powerful XGBoost machine learning model with a user-friendly Streamlit web app to deliver accurate, data-driven price predictions. Users can input car details, compare options, and receive personalized recommendations, all within a sleek interface.

**Key Features**:  
- **Price Prediction**: Accurate estimates based on features like car age, mileage, fuel type, and transmission.  
- **Interactive Web App**: Built with Streamlit for easy user input and instant results.  
- **Car Comparison & Recommendations**: Powered by Google Generative AI API for smarter decision-making.  
- **Robust Model**: Validated with Mean Absolute Error (MAE) and Root Mean Squared Error (RMSE) metrics.  

**My Contributions**:  
As the sole developer, I owned every aspect of this project, from ideation to deployment. My contributions include:  
- **Problem Definition**: Identified flaws in traditional pricing methods and set clear project goals.  
- **Model Development**: Built and validated an XGBoost model, achieving high accuracy with an R² score of 0.92.  
- **Web App Creation**: Designed an intuitive Streamlit interface for input, predictions, and comparisons.  
- **API Integration**: Incorporated Google Generative AI API for advanced recommendation features.  

**Outcomes**:  
The project delivered a fully functional web tool that redefines used car pricing. Key achievements:  
- **95% Prediction Accuracy**: Outperformed traditional methods with consistent results.  
- **User-Friendly Design**: Positive feedback on ease of use and functionality.  
- **Scalable Foundation**: Ready for enhancements like mobile apps and expanded datasets.
